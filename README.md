# oscarTracker

OscarTracker is an Android app written in Java which the purpose is to manage the oscar nominated movies you watched and the ones you didn't and rate each nomination for the movies you watched. 
 
This app was made as a final assignment for the Systems Developing Technologies class by the students [Igor Fran√ßa Negrizoli](https://github.com/igorFNegrizoli) and [Mateus Karvat Camara](https://github.com/MahatKC)
